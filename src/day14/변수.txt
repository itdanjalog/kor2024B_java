- 변수
    - ( 수학 ) 변하는 값
    - ( JAVA ) 하나의 값을 저장할수 있는 메모리 공간
        - ( 변수 타입/분류) 하나의 값/데이터/자료 타입 , 하나의 값을 저장하기 위한 분류/타입 가 존재한다.
            1. 기본타입 ( 리터럴값 자료 ) , 순수한 값
                1. boolean 2.char 3.byte 4.short 5.int 6.long 7.float 8.double
                ex] int a = 10;  # int 타입 변수에 리터럴10 대입

            2. 참조타입 ( 주소값 자료 ) , 값이 위치한 주소(값)
                1. 클래스타입 2. 인터페이스타입 3.배열타입 4.열거타입
                ex] User u1 = new User(); # User 타입 변수에 User객체의 (위치)주소 를 대입

        - 변수 선언 위치에 따라 종류
            1. 지역변수 : 함수/메소드 안에서 선언된 변수 , 선언된 { } 밖으로 호출될수 없다.
            2. 전역변수 : 클래스 안 이면서 함수/메소드 밖에 선언된 타입 앞에 static 키워드가 붙은 변수 , 모든 곳에서 호출이 가능하다.
            3. 매개변수 : 함수/메소드 안으로 들어오는 인자 값을 저장하는 변수 , 함수 안에서만 호출이 가능하다.
            4. (객체)멤버변수 : 클래스 안 이면서 함수/메소드 밖에 선언된 타입 앞에 static 키워드가 없는 변수 , 객체 이용한 호출이 가능하다.
- 상수
    - ( 수학 ) 변하지 않는 값
    - ( JAVA ) 하나의 값을 저장할수 있는 메모리 공간 , 초기화 하고 수정이 불가능
    - 클래스/인터페이스 안 이면서 함수/메소드 밖에 선언된 타입 앞에 final static 키워드가 붙은 변수 -> 상수
